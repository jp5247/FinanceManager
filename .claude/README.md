# Five-Lens Agent Bootstrap

This `.claude` folder was created by a standalone installer.

## First-run checklist

1. Edit `skills/master-skills.md` with your project-specific standards.
2. Review `agents/planner.md` and tune routing keywords for your domain.
3. Review `agents/five-lens-planner.md` context-loading step to match your docs.
4. Run `/five-lens <your system goal>` to generate your first blueprint.
